# 🐍 100 Days of Python Challenge

Welcome to my **100 Days of Python** journey! 🚀
This repository contains my daily progress, where I consistently practice and improve my Python programming skills.

---

## 📌 About This Challenge

The goal of this challenge is simple:

* Code **every day for 100 days**
* Strengthen **Python fundamentals**
* Build **problem-solving skills**
* Document learning through **code + notes**

---

## 📂 Repository Structure

Each day's work is organized into separate Python files:

```
day1.py
day2.py
day3.py
...
day100.py
```

### 🔹 Key Highlights

* Each file represents **one day of learning**
* Code is written based on **topics learned that day**
* **Detailed notes are included as comments inside the code itself**

---

## 🧠 Learning Approach

Instead of maintaining separate notes, I follow this approach:

* 📖 **Concepts explained inside code comments**
* 💡 **Examples included alongside explanations**
* 🧪 **Practice problems implemented directly**
* 🔁 **Revision-friendly structure**

This makes every file:

> A mix of **theory + implementation + practice**

## 🛠️ How to Use This Repo

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/100-Days-of-Python.git
   ```

2. Navigate to the folder:

   ```bash
   cd 100-Days-of-Python
   ```

3. Run any day's file:

   ```bash
   python Day1.py
   ```
